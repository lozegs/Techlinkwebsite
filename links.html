Useful Resources and Links

Use this page to find useful articles, blog posts and websites about coding, women working in this sector and learning to code.

W3Schools

W3Schools is a fantastic resource for both those first starting their coding journey and those wanting tips on some of the more advanced developing techniques. Including help for HTML, CSS, JavaScript, SQL, Python, PHP and Bootstrap, W3Schools is there for all your coding needs.

Coding Blonde

Coding Blonde is a blog dedicated to women working in coding and other technological careers. With weekly newsletters and resources to free coding courses, this site is perfect for women trying to see what it is like to have a career in technology.

The Secret History of Women in Coding

"Computer programming once had much better gender balance than it does today. What went wrong?" This article, written for the New York Times, tells of coding women in the 20th century.

Women in STEM

This website contains many articles showing just how we can break the stereotypes when it comes to females working in the STEM industry. It showcases the different opportunities for women that are available in the science, technology, engineering and maths.

Code First Girls

Want to learn to code? Try their coding kick starter programme. Fancy a change? Try their career switcher programme. Code First Girls offers courses for women and girls to learn how to code.

Four Famous Female Coders You Should Know

Read this article to learn about some of the female coding pioneers you might otherwise have never heard of.

Women in Tech

This website provides a place for women to find jobs in the technology sector, see upcoming events, read guides about working in the industry, and more!

Codecademy

Codecademy is a website which provides lessons for a wide range of coding languages. You can access these lessons for free, or pay a subscription to get extra features, such as quizzes and real-world projects.

'Every Girl Has to Learn How to Code'

An article written by Mariah Espada, featuring Reshma Saujani, the CEO of Girls Who Code.
